# Sparky Engine 2
A powerful 3D game engine built with C++ and Vulkan, featuring a test platformer game.

- **VulkanRenderer**
- **WindowManager**: 
- **InputManager**
- **Camera**
- **GameEntity**

### Prerequisites
- C++17 compatible compiler
- CMake 3.20 or higher
- Vulkan SDK
- GLFW library
- GLM mathematics library

## Controls
- **WASD**: Move player
- **Space**: Jump
- **Mouse**: Look around
- **ESC**: Exit the game
